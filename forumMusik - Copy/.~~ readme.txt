1. You need XAMPP from https://www.apachefriends.org/ for laravel project
2. After XAMPP installed, go to htdocs on your XAMPP folder and copy/clone the project there
3. Read laravel documentation on https://laravel.com/docs/4.2/quick, for installation
4. After that, create new database on phpMyAdmin for the app
5. Match the ".env" with your database name & username("root")
6. Type "php artisan migrate" for migrate available data on the project and "php artisan db:seed" for seed data
7. Activate Apache & MySQL on XAMPP then go to your terminal from Visual Studio Code and type "php artisan serve"
8. Open the development server and you can use the forum..
